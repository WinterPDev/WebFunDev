Position Static

The default value for the position property. Top, bottom, left, and right will have NO effect.

Position Relative

The element is positioned relative to its normal position. Top, bottom, left, and right have effect.

Position Absolute

The element is positioned relative to it's closest positioned ancestor. Top, bottom, left and right have effect.

Position Fixed

The element is positioned relative to the browser window. Top, bottom, left, and right have effect.

Position Sticky

The element is positioned relative to the user's scroll position. Top, bottom, left, and right have effect.
